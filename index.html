⚙️ Configuration Back Office

Pour synchroniser le Back Office avec ton app iPhone, crée un espace de stockage cloud gratuit sur JSONBin.io en suivant ces étapes :

1
Va sur jsonbin.io → Sign Up gratuit
2
Clique "+ New Bin" → colle ce JSON : {"experts":[],"content":[],"modes":{},"quickBtns":[]} → Create Bin
3
Copie le BIN ID affiché (longue suite de chiffres/lettres)
4
Va dans ton profil JSONBin → API Keys → copie ta clé (commence par $2b$)
